Repository Naming Rules:

1. Use lowercase letters only.
2. Use hyphens (-) instead of spaces.
3. Name must clearly indicate the project.

Format:
topic-name-purpose

Examples:
- python-basics-practice
- data-cleaning-project
- eda-sales-dataset
- student-performance-analysis
- placement-aptitude-practice

Do NOT use:
- random names
- personal names
- version numbers (v1, v2)
- words like final, new, test
